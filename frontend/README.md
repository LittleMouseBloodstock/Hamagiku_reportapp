# Shinba Report Frontend

Shinba Report is an owner reporting SaaS for racehorse operations. It supports multilingual PDF report creation, client management, and AI-assisted comments for owners and clients.

## Development

Run the local development server:

```bash
npm run dev
```

Open `http://localhost:3000`.

## Key pages

- `/` public landing page
- `/login` sign-in page
- `/dashboard` authenticated app
- `/dashboard/billing` subscription management
- `/legal/terms`
- `/legal/privacy`
- `/legal/refund`
